
# Home Network IDS - Raspberry Pi 4 + Suricata 7

## Overview
Raspberry Pi 4 running Suricata 7 IDS with Emerging Threats ruleset, 
monitoring home network traffic and generating alerts in JSON format.

## Hardware
- Raspberry Pi 4 Model B
- Connected via ethernet to home network switch

## Stack
- Raspberry Pi OS Lite 64-bit
- Suricata 7.0.10
- Emerging Threats Open Ruleset (50,143 rules)

## Status
- [x] Suricata installed and running
- [x] Rules loaded and verified
- [x] First alert generated
- [x] Python alert parser
- [ ] AI-assisted triage
